# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.
The game allows users to play against the computer with live score updates and dynamic result messages.

---

## 🚀 Features

* 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
* Random computer-generated choices
* Live score tracking
* Win / Lose / Draw result display
* Interactive hover effects
* Responsive layout using Flexbox
* Beginner-friendly and clean code structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
